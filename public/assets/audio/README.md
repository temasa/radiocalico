# Audio Assets

Place your audio files in this folder to play them on the Radio Calico website.

## Supported Formats
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## Usage
1. Add your audio files to this folder
2. Update the playlist in `app.js` to reference your files

## Example
If you add a file named `stream.mp3` to this folder, it will be accessible at:
`/assets/audio/stream.mp3`
